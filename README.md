# metalmax
Full C / C ++ / arm-v8 assembly remake metalmax

用c c++ arm-v8汇编重写的 重装机兵，目标Android平台
图像使用opengl es 2，通过alpha方式生成2D贴图实现硬件加速
音频使用opensl es，暂时使用nsfplayer生成PCM数据传给opensl播放，后续计划自建音频引擎，分离音频数据
